바이빗 API키 발급 방법

   API키 관리화면에서 「Create New Key」를 클릭한후 「System-generated API Keys」를 선택합니다..

   아래와 같이 입력하여 「Submit」를 클릭한후 생성된 API키와 시크릿키를 복사하여 API등록화면에 등록해 주세요.
   출금은 절대 체크하지 마세요!    시크릿키는 화면을 닫으면 확인할수 없으니 반드시 메모해 두세요.

바이낸스 API키 발급 방법

   API키 관리화면에서 라벨명을 기입한후 「Create API」를 클릭하면 아래와 같은 화면이 나타납니다.

   「Edit」를 클릭하여 위와 같이 입력한후 생성된 API키와 시크릿키를 복사하여 API등록화면에 등록해 주세요.
   출금은 절대 체크하지 마세요!
   시크릿키는 화면을 닫으면 확인할수 없으니 반드시 메모해 두세요.
   현물과 선물 둘다 사용할 경우에는 반드시 아래와 같이 동일한 API키를 현물과 선물에 입력하여 등록해 주세요.

   위에서 메모한 ①②③을 아래와 같이 API키 등록화면에 입력하여 등록합니다.

빗썸 API키 발급 방법

   API 관리화면에서 아래의 항목을 선택한후 API KEY 생성」을 클릭하여 API를 생성합니다.

   생성된 API에서 활성화 버튼을 클릭하여 승인을 합니다.
   승인된 API키와 시크릿키를 복사하여 API등록화면에 등록해 주세요.
   출금은 절대 체크하지 마세요!

업비트 API키 발급 방법

   업비트 로그인 – 고객센터 – Open API 안내 – Open API 사용하기를 순서대로 클릭해 주세요.

   자산조회,주문조회,주문하기를 체크하고 봇의 IP주소를 입력한후 Open API Key 발급받기를 클릭합니다.    Access키와 Secret키를 복사하여 확장프로그램의 API키 등록화면 업비트란에 붙여쓰기를 합니다.    시크릿키는 화면을 닫으면 확인할수 없으니 반드시 메모해 두세요.

OKX API키 발급 방법

   아래의 화면에서 ②와 ③을 메모한후 확인버튼을 클릭합니다.    위에서 메모한 ①②③을 아래와 같이 API키 등록화면에 입력하여 등록합니다.

한국투자증권 API키 발급 방법

한국투자증권 로그인
   다음 주소를 통해 한국투자증권 홈페이지에 접속합니다. http://www.truefriend.com   ※우측 상단의 로그인 메뉴를 선택합니다.

   스마트폰 인증을 선택합니다. 스마트폰에 설치된 한국투자증권 앱에서 [메뉴]-[인증센터]-[PC인증]을 선택한 후 웹 브라우저에 표시된 QR 코드를 스캔하거나 인증번호를 입력하면 인증이 완료되고 자동으로 로그인됩니다.
KIS Developers 서비스 신청하기
   한국투자증권 홈페이지 상단의 메뉴에서 [트레이딩]-[Open API]-[KIS Developers]를 선택합니다. 인증번호 요청을 클릭하면 문자로 인증 번호가 전송됩니다. 인증번호를 입력한 후 '신청'을 클릭합니다.
   유의사항확인 단계에서 약관에 '동의'를 합니다.
   신청정보 단계에서 여러분의 개인 정보를 확인합니다. 아이디는 다음 단계에서 사용되므로 잘 확인해두시기 바랍니다. 계좌 정보에서 앞서 계설한 계좌를 연결합니다.
   KIS Developers 신청 완료 화면이 나타납니다. 발급된 APP Key와 APP Secret 값을 복사한 후 따로 저장해둡니다. 이 값들을 사용하여 한국투자증권의 API를 사용할 수 있기 때문에 유출에 주의해야합니다.
모의 투자 신청하기
   K오픈 API를 모의투자 계좌로 사용하려면 먼저 모의투자 신청을 해야합니다. 한국투자증권 홈페이지에서 [트레이딩]-[모의투자]-[주식/선물옵션 모의투자]-[모의투자안내]로 이동한 후 모의투자를 신청합니다. 리그구분에서 국내주식 및 해외주식을 선택합니다.
   한국투자증권 홈페이지 상단의 메뉴에서 [트레이딩]-[Open API]-[KIS Developers]를 선택합니다. 인증번호 요청을 클릭하면 문자로 인증 번호가 전송됩니다. 인증번호를 입력한 후 '신청'을 클릭합니다. KIS Developers 서비스 신청하기에서 모의투자계좌를 선택한 후 앞서 신청한 모의투자 계좌를 입력합니다.
   오픈 API는 실전투자계좌와 모의투자계좌 각각 APP Key, APP Secret이 생성됩니다.
TVExtBot에 API키 등록하기
   TVExtBot의 API키 등록화면에서 계좌번호, APP Key, APP Secret 순으로 입력해서 등록하면 됩니다.
        ※ 계좌번호는 반드시 숫자 10자리가 필요합니다. 예) 50071022-01 또는 5007102201
   실전계좌는 Kis 에 모의계좌는 Kis-Testnet 에 등록해 주세요.
