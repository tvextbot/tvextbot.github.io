바이빗 API키 발급 방법

   API키 관리화면에서 「Create New Key」를 클릭하면 아래와 같은 화면이 나타납니다.

   위와 같이 입력하여 「Submit」를 클릭한후 생성된 API키와 시크릿키를 복사하여 API등록화면에 등록해 주세요.
   시크릿키는 화면을 닫으면 확인할수 없으니 반드시 메모해 두세요.
   출금은 절대 체크하지 마세요!

바이낸스 API키 발급 방법

   API키 관리화면에서 라벨명을 기입한후 「Create API」를 클릭하면 아래와 같은 화면이 나타납니다.

   「Edit」를 클릭하여 위와 같이 입력한후 생성된 API키와 시크릿키를 복사하여 API등록화면에 등록해 주세요.
   API등록화면에서 동일한 API키를 현물과 선물에 각각 입력하여 등록해 주세요.
   시크릿키는 화면을 닫으면 확인할수 없으니 반드시 메모해 두세요.
   출금은 절대 체크하지 마세요!

비트겟 API키 발급 방법

   Bitget 거래소에서 API만들기 버튼을 클릭하여 아래의 내용을 입력후 다음을 클릭합니다. (①을 메모)

   아래의 화면에서 ②와 ③을 메모한후 확인버튼을 클릭합니다.    시크릿키는 화면을 닫으면 확인할수 없으니 반드시 메모해 두세요.
   위에서 메모한 ①②③을 아래와 같이 API키 등록화면에 입력하여 등록합니다.

빗썸 API키 발급 방법

   API 관리화면에서 아래의 항목을 선택한후 API KEY 생성」을 클릭하여 API를 생성합니다.

   생성된 API에서 활성화 버튼을 클릭하여 승인을 합니다.
   승인된 API키와 시크릿키를 복사하여 API등록화면에 등록해 주세요.
   출금은 절대 체크하지 마세요!
